# v0.24.7 - 2026-06-14

## 前情与承接

`v0.24.7` 承接 `v0.24.6` 的 B站工具集成与 Agent 可靠性改进，本版本聚焦**从 akashic-agent 移植生产级 HTTP 基础设施**——akashic 的 `core/net/http.py` 提供了指数退避重试、请求预算、分级连接池等完整能力，CaiBao 的 web_fetch/web_search 此前缺乏重试机制，遇到 429/5xx/网络抖动直接失败。

## 版本概览

1. **HTTP 重试基础设施** — 从 akashic-agent 移植 `RetryPolicy` + `RequestBudget` + `retry_request()`，同步化适配
2. **web_fetch/web_search 接入重试** — 4 个 HTTP 调用点（web_fetch / Exa / Brave / Tavily）全部接入
3. **工具发现增强** — `ToolDefinition` 新增 `search_hint` 字段，LLM 可用自然语言发现工具

## 新增功能

### HTTP 重试基础设施 (`app/services/net/http.py`)

从 akashic-agent `core/net/http.py` 移植，同步化适配：

**RetryPolicy** — 可配置重试策略：
```
max_attempts: 3          # 最多 3 次尝试（含首次）
retry_statuses: 408, 429, 500, 502, 503, 504    # 可重试状态码
base_delay_s: 0.3        # 基础退避 300ms
max_delay_s: 1.5         # 退避上限 1.5s
jitter_ratio: 0.2        # ±20% 随机抖动
```

**RequestBudget** — 跨重试绝对截止时间：
- web_fetch: `timeout + 15s`（如用户设 30s 超时，总预算 45s）
- Brave/Tavily: 默认 30s（2×15s 单次超时）
- Exa: 默认 50s（2×25s 单次超时）

**预置策略**：
- `RETRY_STANDARD`：3 次尝试，300ms 基退避，1.5s 上限
- `RETRY_FAST`：2 次尝试，150ms 基退避，300ms 上限
- `RETRY_NONE`：单次尝试，不重试

**退避算法**：`base_delay × 2^(attempt-1) + random(-jitter, +jitter)`，钳制在 `[0, max_delay]`

## 主要变更

### web_fetch / web_search 接入重试

| 调用点 | 旧实现 | 新实现 |
|--------|--------|--------|
| `web_fetch_handler` | `with httpx.Client(...) as client: client.get(url)` | `retry_request(client, "GET", url, budget=...)` |
| `_exa_search` | `httpx.post(url, timeout=25.0)` | `retry_request(client, "POST", url, timeout_s=25.0)` |
| `_brave_search` | `httpx.get(url, timeout=15)` | `retry_request(client, "GET", url, timeout_s=15.0)` |
| `_tavily_search` | `httpx.post(url, timeout=15)` | `retry_request(client, "POST", url, timeout_s=15.0)` |

B站相关函数（`_request_bilibili_json` 等）暂不接入——它们已有独立的多级降级策略，共用 `httpx.Client` 实例，打法不同。

### ToolDefinition 新增 search_hint

允许 LLM 通过自然语言发现和选择工具：

```python
@dataclass(frozen=True)
class ToolDefinition:
    ...
    search_hint: str = ""  # 新增字段，e.g. "搜索 查资料 谷歌 Bing"
```

已填充的 hint：
| 工具 | search_hint |
|------|-------------|
| `web_search` | `搜索 查资料 谷歌 Bing` |
| `web_fetch` | `读取网址 浏览网页 抓取内容` |
| `bilibili_latest_videos` | `B站 bilibili UP主 最新视频 投稿` |

`to_dict()` 同步输出该字段。
